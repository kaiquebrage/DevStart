## 💡 Sobre o Projeto

O **DevStart** é uma landing page completa que simula uma plataforma real de venda de cursos digitais.

O projeto foi desenvolvido com foco em:

- Experiência do usuário (UX)
- Conversão (Copywriting)
- Interatividade com JavaScript
- Boas práticas de desenvolvimento Front-End

---

## 🎯 Objetivo

Criar uma aplicação front-end funcional que represente um produto real do mercado, indo além de um layout estático.

---

## 🧠 Diferencial do Projeto

Este projeto se destaca por unir:

💻 Desenvolvimento Front-End  
📈 Estratégia de Marketing  
🎯 Foco em conversão  

> Interfaces que não apenas funcionam, mas geram resultado.

---

## 🛠️ Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3 (Flexbox, Grid e Responsividade)
- JavaScript (DOM, eventos e interatividade)

---

## ⚙️ Funcionalidades

- ✅ Navegação entre páginas
- ✅ Menu com estado ativo automático
- ✅ Menu animado com indicador (UX avançada)
- ✅ Scroll suave até seções
- ✅ Formulários com validação
- ✅ Modal de confirmação de compra
- ✅ Feedback visual dinâmico
- ✅ Layout responsivo (mobile first)

---

## 🎨 Experiência do Usuário

O projeto foi pensado para simular uma jornada real:

1. Atração (Hero + Copywriting)
2. Interesse (Benefícios)
3. Decisão (Cursos)
4. Prova social (Depoimentos)
5. Conversão (Formulário + CTA)
